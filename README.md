# Chat via Socket.IO

## Como Iniciar

1) Clone este repositório e abra um terminal na pasta clonada;

2) Execute o seguinte comando para instalar as dependências:
```shellscript
> npm install
```

3) Execute o seguinte comando para iniciar a aplicação:
```shellscript
> npm start
```
